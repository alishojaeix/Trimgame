# Simple Python Trivia Game


A simple trivia game built with Python and Pygame that uses online images for questions.

## Features

- Multiple-choice questions with images
- Score tracking
- Responsive design
- Fallback when images fail to load

## Requirements

- Python 3.6+
- Pygame
- Requests
- Pillow (PIL)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/alishojaeix/trivia-game.git
   cd trivia-game
   ```

2. Install dependencies:
   ```bash
   pip install pygame requests pillow
   ```

## How to Play

- Run the game: `python trivia_game.py`
- Click on the correct answer for each question
- See your final score at the end
- Press 'R' to restart the game

## Customization

Easily modify the game by:
- Adding more questions to the `questions` list
- Changing the image URLs
- Adjusting the visual appearance
